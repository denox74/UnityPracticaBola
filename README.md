# Unity Practica - Roll a Ball

ES - Español
------------

Descripción
- Proyecto de práctica con Unity: "Roll a Ball".
- Controlas una bola, recoges pickups y evitas enemigos.

Requisitos
- Unity (recomendado 2020.x - 2021.x).
- Git.

Instalar desde el repositorio (clonar)
1. Abre una terminal (PowerShell o CMD) y ejecuta:
   git clone https://github.com/denox74/UnityPracticaBola.git
2. Abre Unity Hub → Add → selecciona la carpeta clonada.
3. Abre el proyecto en Unity.

Cómo ejecutar
1. En el Editor de Unity, pulsa Play para probar.
2. El botón Start en la UI llama a StartLevel() para habilitar jugador y enemigos.

Controles
- Movimiento: WASD o flechas.
- Saltar: Space (si estás en el suelo).
- UI Start: inicia la fase.

Notas rápidas
- Scripts principales: Assets/Scripts/PlayerController.cs y Assets/Scripts/StartBotton.cs
- Asigna el método StartLevel() al OnClick del botón en el Inspector.
- Revisa la consola de Unity para mensajes/errores.

Cómo construir (exportar)
1. File > Build Settings.
2. Añade la(s) escena(s) a "Scenes In Build".
3. Selecciona plataforma y Build.

Git - pasos básicos para contribuir
1. git clone https://github.com/denox74/UnityPracticaBola.git
2. git checkout -b nombre-rama
3. git add .
4. git commit -m "Descripción del cambio"
5. git push origin nombre-rama
6. Crear Pull Request en GitHub

EN - English

Description
- Unity practice project: "Roll a Ball".
- Control a ball, collect pickups and avoid enemies.

Requirements
- Unity (recommended 2020.x - 2021.x).
- Git.

Install from the repo (clone)
1. Open a terminal (PowerShell or CMD) and run:
   git clone https://github.com/denox74/UnityPracticaBola.git
2. Open Unity Hub → Add → select the cloned folder.
3. Open the project in Unity.

How to run
1. In the Unity Editor press Play to test.
2. The Start UI button calls StartLevel() to enable player and enemies.

Controls
- Move: WASD or arrow keys.
- Jump: Space (when grounded).
- Start UI: begin the level.

Quick notes
- Main scripts: Assets/Scripts/PlayerController.cs and Assets/Scripts/StartBotton.cs
- Assign StartLevel() in the Button OnClick (Inspector).
- Check Unity Console for errors/messages.

Build (export)
1. File > Build Settings.
2. Add scenes to "Scenes In Build".
3. Choose platform and Build.
